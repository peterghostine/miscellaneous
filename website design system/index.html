Color Schemes

The University of Toronto conducted an interesting study on colors and how they are perceived by individuals. They determined that most people prefer combinations of simple colors. In most cases, just two or three colors were perceived as appealing. That’s why sticking to a color palette is so important to the success of your site, and ultimately your business. Read more...

Text Styles

  • <H1> = Heading Level 1

    • Font family: Roboto
    • Font size: 32px
    • font weight: 1000

  • <H2> = Heading Level 2

    • Font family: Roboto
    • Font size: 24px
    • font weight: 800

  • <H3> = Heading Level 3

    • Font family: Roboto
    • Font size: 19px
    • font weight: 600

  • <H4> = Heading Level 4

    • Font family: Roboto
    • Font size: 16px
    • font weight: 500
  • <H5> = Heading Level 5
    • Font family: Roboto
    • Font size: 16px
    • font weight: 500
  • <H6> = Heading Level 6
    • Font family: Roboto
    • Font size: 16px
    • font weight: 500

  • <p> = Paragraph

    • Font family: Oxygen
    • Font size: 16px
    • font weight: 400
  • List
    • Font family: Open Sans Condensed
    • Font size: 18px
    • font weight: 400

Note: every style name is displayed in its corresponding style (per the bulleted list). The bulleted list itself is displayed using the 'list' style.
